# Solana-TPS-Monitor
Bash script to monitor solana TPS

# Run on Terminal 

git clone https://github.com/fallenangel2305/Solana-TPS-Monitor.git

# on mac 

brew install jq

cd Solana-TPS-Monitor

sh solana_tps.sh

# on ubuntu

sudo apt update
sudo apt install jq

cd Solana-TPS-Monitor

sh solana_tps.sh
